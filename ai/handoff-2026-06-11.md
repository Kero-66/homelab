## Completed This Session

### 1. Jellyfin Subtitle Auto-Select Fixed
- **Was**: `SubtitleMode=Default`, `SubtitleLanguagePreference=""` — user had to manually pick subs every episode
- **Now**: `SubtitleMode=Smart` (enum=4), `SubtitleLanguagePreference=eng`
- Smart mode: auto-shows English subs when audio is non-English; silent on English-audio shows
- API: `POST /Users/{id}/Configuration` with integer enum (not string) for SubtitleMode

### 2. Full Subtitle Audit
- Queried Jellyfin `/Users/{id}/Items?IncludeItemTypes=Episode&fields=MediaStreams` for all episodes
- 74 shows missing English subs (after excluding English-language shows)
- Priority shows identified: JoJo (213/214), AoT (123/131), Monogatari (120/122), Golden Kamuy (109/124), Overlord (75/114), Frieren (66/66), etc.

### 3. Bazarr Providers Expanded
- Added **jimaku** and **subsource** via UI (Settings → Providers) — no credentials needed
- Current providers: animetosho, bsplayer, gestdown, jimaku, subf2m, subsource
- **Key constraint**: `enabled_providers` is silently ignored by Bazarr API (both read and write); config.yaml is regenerated from DB on restart — only the UI works
- Documented in `media/AGENTS.md`

### 4. Bazarr Searches Triggered
- Lord of Mysteries S01 E01–E04, E06, E13: `PATCH /bazarr/api/episodes/subtitles?seriesid=16&episodeid=X&language=en&forced=false&hi=false`
- Terror Man S01 E01–E08: same pattern, seriesid=108, episodeids=4071–4078
- LoM E01 (animetosho), E02 (subf2m), E03 (animetosho), E13 (animetosho) confirmed downloaded
- LoM E04, E06 and Terror Man E1–E8: still in wanted list (not found by providers)

### 5. Multi-Subs Custom Format Added to Sonarr + Recyclarr
- **Root cause**: Sonarr was grabbing KOR-only releases (no subs) because multi-sub fansub releases scored 0, rejected as "not an upgrade" vs existing file at score 300
- Created Sonarr CF "Multi-Subs" (id=289), regex: `(?i)(multi[\-. ]?sub|msubs?|\[multi\])`
- Scored 350 on Anime (1080p) profile (id=14) — beats existing ToonsHub files at 300
- Added to recyclarr for persistence: `truenas/stacks/recyclarr/custom-formats/multi-subs.json` (trash_id: `homelab-multi-subs`)
- Updated `recyclarr.yml` to reference it with score 350 across all three profiles
- Added `resource_providers` to `settings.yml` on TrueNAS pointing to `/config/custom-formats`
- **SeriesSearch triggered** for Terror Man (seriesId=108, command=519626) — should now grab LuckFansubs multi-sub releases
